# Icons

このディレクトリには、以下のアイコンを配置してください：

## 必要なアイコンファイル

- `icon16.png` - 16x16ピクセル
- `icon48.png` - 48x48ピクセル
- `icon128.png` - 128x128ピクセル

## アイコンの作成方法

1. Misskeyのロゴやオリジナルのデザインを使用してアイコンを作成
2. 各サイズにPNG形式で書き出し
3. このディレクトリに配置

## 一時的な代替手段

アイコンがない場合、以下の方法でプレースホルダーを作成できます：

### オンラインツールを使用

- [Favicon.io](https://favicon.io/) - テキストからアイコンを生成
- [Real Favicon Generator](https://realfavicongenerator.net/) - 様々なサイズのアイコンを生成

### シンプルなSVGから生成

簡単なカラーブロックや文字をSVGで作成し、PNGに変換することもできます。

## ライセンスについて

Misskeyのロゴを使用する場合は、Misskeyのブランドガイドラインに従ってください。
